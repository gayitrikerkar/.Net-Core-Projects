# Internal dependencies

The project uses internal libraries, that are provided by the NuGet-Server of [Azure DevOps Servers](https://pa-tfs.patec.group/Col-PA/). The NuGet source for this Feed is added within the Visual-Studio solution itself
and requires no manual intervention. The Links in the following list require Access to the Intranet of Syntegon and Read-Access to the Azure DevOps Server.

## Used libraries

- [Syntegon.Tools.MessageLoggerCore](https://pa-tfs.patec.group/Col-PA/Cr_PA-PH_Software-Tools/_git/MessageLogger)
